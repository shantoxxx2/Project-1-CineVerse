# 🎬 CineVerse - Movie Streaming Platform

![CineVerse Banner](https://via.placeholder.com/1200x400/8e44ad/ffffff?text=CineVerse)

## 📖 Overview

**CineVerse** is a modern movie and TV show streaming platform UI inspired by popular services like Netflix and Prime Video.
It features a **clean, responsive design**, dynamic movie browsing, and a modern dark-themed interface.

The project demonstrates how a streaming dashboard can be built using **pure HTML, CSS, and JavaScript** without external frameworks.

---

## ✨ Features

* 🎥 **Modern Streaming UI**
* 📱 **Fully Responsive Design (Desktop / Tablet / Mobile)**
* 🔎 **Search Functionality**
* 🎞️ **Dynamic Movie Grid**
* 📂 **Category Filtering**
* 🔥 **Trending Movies Section**
* ⭐ **Featured Content Area**
* ➕ **Load More Movies Button**
* 📱 **Mobile Friendly Navigation**
* 🎬 **Streaming Video Player Support**
* 🎯 **Smooth Animations and Hover Effects**

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/cineverse.git
```

### 2️⃣ Open the Project

Navigate to the project folder:

```bash
cd cineverse
```

### 3️⃣ Run the Website

Simply open **index.html** in your browser.
